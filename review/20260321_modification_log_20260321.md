## 📝 文字层面需要改进的地方（尚未执行）

### 通用规则（适用于所有章节）

1. **删除模糊用语**:
   - "显然" → "可以直接验证"或删除
   - "易知" → 补充简要步骤
   - "不难得到" → 给出关键中间步骤

2. **首次出现的专业术语**:
   - 加粗并标注英文，例如：**布洛赫定理**（Bloch theorem）

3. **所有公式后必须加物理解释**:
   ```latex
   \begin{equation}
     g_{\mathrm{th}} = \alpha_{\mathrm{rad}} + \alpha_{\mathrm{int}}
   \end{equation}
   其中$\alpha_{\mathrm{rad}}$代表辐射损耗（光子从腔中逸出的速率），$\alpha_{\mathrm{int}}$代表内部损耗（吸收、散射等非辐射过程）。该式的物理意义是：阈值增益必须恰好补偿所有损耗机制的总和。
   ```

4. **补充数量级**:
   -  wherever提到典型值、范围，必须给出具体数字
   - 示例："高温工作" → "高温工作（$T > 80^\circ\mathrm{C}$）"
