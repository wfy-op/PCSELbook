## 一、全书总目标

这本书必须实现以下目标：

1. 面向“有基础电动力学/光学/半导体激光器背景，但从未系统学过 PCSEL”的读者。
2. 让读者顺着一本书连续阅读，就能建立对 PCSEL 所需理论、仿真、器件物理、工作流与验证逻辑的整体理解。
3. 同时做到 **pedagogical** 与 **rigorous**：所有关键结论必须给出来源，要么完整推导，要么说明推导骨架、近似条件、适用边界与不能做的外推。
4. 全书必须是**教科书**，不是散乱讲义，不是堆砌文献综述，不是软件按钮教程，也不是纯数学笔记。
5. 最终结果必须是**可编译的 LaTeX 全书工程**，能够生成 PDF；需要时使用 TikZ/PGFPlots 绘图；必要时给出可运行的 Python 参考脚本或伪代码。
6. 全书必须始终守住层级边界：  
   - 被动模 ≠ 阈值模  
   - 阈值增益 ≠ 阈值电流  
   - 无限周期结果 ≠ 有限器件工作结果  
   - 均匀增益 ≠ 真实电注入分布  
   - 冷腔结论 ≠ 器件工作结论

## 五、写作风格与叙事要求

### 1. 语言与术语
- 全书使用**简体中文**。
- 术语第一次出现时给出**中英对照**，例如“布洛赫定理（Bloch theorem）”。
- 后续以中文为主，必要时保留标准英文缩写。
- 不要混入未本地化残留提示词，如 `chapter/chapters/eq./fig.`。

### 2. 风格
- 像优秀研究生教材，不像聊天、不像宣传文、不像 PPT 讲稿。
- 句子要稳、准、可追踪。
- 尽量避免模糊判断词；若结论依赖近似，必须显式写出近似。

### 3. 叙事方式
必须遵循以下教学顺序：

- 先给物理问题
- 再给直觉图像
- 再给定义、坐标、变量、量纲、边界条件
- 再给数学形式与推导骨架
- 再解释这个结果在物理上意味着什么
- 再说明它能算什么、不能算什么
- 再给例子、易错点、与后续章节的连接

### 4. 必须避免的写法
- 主题清单式堆砌
- 词条式解释代替理论展开
- 各章彼此独立、互不依赖
- 公式罗列而不解释来源
- 只讲现象不讲推导
- 只讲推导不讲物理图像
- 只讲纯光学，不讲外延层、电注入、热效应
- 把 PCSEL 写成普通光子晶体共振器
- 把冷腔结果直接当器件结论
- 用空泛套话填充篇幅

- 明确写出“需要材料数据库 / 实验校准 / 工艺窗口支持”
- 或在工程中使用 `TODO` 标明待核实项
- 若只是解析示意图、数值示意或文献复绘，必须明确注明来源性质

若结果只是近似、启发式、经验式、渐近式或数值观察，必须明示，不得偷换成严格结论。

## 十九、禁止事项

- 不要写成百科式堆砌
- 不要把软件按钮教程当核心内容
- 不要用虚构仿真图替代真正可解释的图
- 不要为了“看起来全面”而牺牲逻辑顺序
- 不要把高 Q、BIC、Γ 点、带边、低阈值、大面积单模混成一句口号
- 不要把“术语解释”误当成“理论完成”
- 不要把二维标量模型外推成真实三维 slab 器件结论
- 不要把无限周期单胞结果偷换为有限器件性能
- 不要把冷腔本征模排序直接等价为阈值电流排序

## 二十、输出阶段要求

### 第一阶段：先做全书级规划
在开始正式写正文之前，必须先输出并等待继续：

1. 全书定位
2. 目标读者画像
3. 总目录（建议分 Part）
4. 每章一句话任务说明
5. 符号系统设计
6. 前后依赖图

9. 建议的全书 Part-章结构
10. 主线问题链
11. 每章输出给下一章什么
12. 图表清单
13. 全书必须反复守住的十大红线

### 第二阶段：输出完整 LaTeX 工程结构
必须给出：
- `main.tex`
- `frontmatter/`
- `parts/`
- `chapters/`
- `appendices/`
- `figures/`
- `code/`
- `bib.bib`
- `Makefile` 或 `latexmk` 构建说明

### 第三阶段：按章写作
按章写作时必须始终维护全局一致性：
- 统一术语
- 统一符号
- 统一交叉引用
- 统一近似边界
- 统一图号表号章号

### 第四阶段：全书总审校
完稿后必须执行一次全书总审校，检查：

1. 章节顺序是否最优
2. 是否有 stale chapter reference

- 若某些图需要数据，必须说明其来自解析示意、计算结果还是文献复绘

## 二十一、执行时的自检问题

在写作过程中，始终反复自检：

1. 这一段是不是在真正教读者，而不是在列清单？
2. 这一章是不是自然依赖前一章，并推动后一章？
3. 这一结论是不是明确说明了模型边界？
4. 这一仿真结果是不是被错误地上升为器件结论？
5. 外延层、电注入、热效应是不是被真正纳入主线？
6. 初学者是否可以顺着本书从零走到器件级理解？
7. 图、表、公式是不是承担了教学任务，而不是装饰？
8. 本章是不是交代了“能算什么、不能算什么”？
9. 是否出现了把被动模、阈值模、器件模混为一谈的风险？
10. 是否出现了把单点最优误当成工作窗口最优的风险？

如果其中任一问题答案是否定，就需要继续改写，而不是继续堆内容。
